# Disaster-Rescue-Route-Planner
The Disaster Rescue Route Planner is a graph-based C program designed to compute the most optimal rescue paths during emergency situations such as natural disasters. It helps identify the shortest and safest routes from a rescue base to multiple affected locations using Dijkstra’s Shortest Path Algorithm.  

Features
- Computes shortest rescue routes from a source location
- Displays complete optimal path for each destination
- Estimates rescue time based on distance
- Classifies risk levels (Low, Medium, High)
- Generates structured rescue report output
- Uses adjacency matrix for graph representation
